Images

agar hame koi bhi image apne website pe lagani hai to ham uske liye 'img' tag ka use karte hai. is tag ko lagate hi tumko dikhega 2 attriutes 'src' aur 'alt'. src me value dalegi loaction of the image and alt me aap dalenge ek text jooki image ke unavailable hone pe dikhai dega.








agar aapko image ki height ya width set karna hai to aap wo bhi kar sakte hai width/height ka use karke. usme aap apne hisab se width/ height set kar sakte hai jo ki ek value daali jayegi. default unit height/width ki pixels hai jo ki agar nahi bhi daali jaaye to bhi acceptable hai.

Tables

Hame agar tables show karni hai website pe to uske liye 'table' tag use karenge. table tag ke andar aate hai 'tr' aur 'td'. tr stands for table row, whereas td stays for table data. tr is used to put in values in a row. tr ke andar aata hai td jisme values dalti hai table ke. agar aapko table head daalna hai to uske liye table ka pehla tr me jo bhi table heading likhna hai usse 'th' ke andar likhe.
Example -


S.No. Name Phone
1 Ayush 99776355
2 Sukomal 85767983



Jaisa ki aapne dekha ki table jo hai uske around borders agaya hai to wo aap style attribute ka use karke kar sakte hai. Border wagera ke baare me ham aage janenge. Tab tak ke liye aap inspect karke dekh sakte hai ki table ki jo styling ki gayi hai wo kaise hui hai.

Colspan and Rowspan

Colspan and rowspan jo hai wo aap tab use kar sakte hai jab aapko kisi ek data ko kuch rows ya kuch columns ki jagah deni hai. example ke taur pe maanlo ki 1 aadmi ka ek hi phone number hai to uske liye aap colspan use karenge, whereas, agar ek aadmi ke paas 2 alag phone numbers hai to aap rowspan use kar sakte hai.


S.No. Name Phone AltPhone
1 Ayush 99776355 66377335
2 85767983 48385645
3 Sukomal 87456233



Captions

Captions jo hai wo table kis baare me hai wo batata hai, basically table ka naam hai.

Headers, Footers And Body

Table ke andar bhi header, footer aur body hoti hai, aur wo generaly developer pe hota hai ki wo konsa part table ka kisme daale. isse visually to kuch change nahi aata table me but ye chize organise rakhne me kaafi help aata hai.

Lists

Ye teeen type ke hoti hai, unordered, ordered and Definition list. Lists hame kuch bhi list banane me help karti hai.

Unordered Lists

Ye jo list hoti hai wo generally koi order follow nahi karti. Ise ul tag use karke likha jaata hai. Example -
  • Apple
  • Mango
  • Banana
  • Guava

Ordered Lists

Ye jo list hoti hai wo number wise listed hoti hai. Ise ol tag use karke likha jaata hai. Example -
  1. Apple
  2. Mango
  3. Banana
  4. Guava

Definition Lists

Ye jo list hoti hai wo kisi ki jab definition likhni ho tab aap use karte hai. Ise dl tag use karke likha jaata hai. dl tag ke andar aate hai 2 aur tags, dt and dd. dt stands for definition term and dd stands for define definition. Example -
Apple
A red color fruit.
Mango
A yellow color fruit also known as King of fruits.



Special Note - Agar aapko unordered list and ordered list me change karna hai ki wo list kis way me show ki jaaye to wo aap type attribute ka use karke kar sakte hai. Ye type attribute aapko ul ya ol tag ke saath likhna hoga.
Types of Unordered List - Disc, Square, Circle.
Types of ordered lists - i,I,A,a,1
