ID

ID is an attribute for any element in HTML. ye ek unique identifier hai jo sirf ek hi elemnt pe lagg sakta hai. iska use unique styling and js manipulations ke liye kiya jaata hai.
Example ke liye agar ham iss html page pe css laga lete hai aur let's say ki ye jo paragraph chal raha hai ispe ham id lagana chahe to hame bas iss paragraph ke p tag me 'id' likhna hoga aur uski value aap ek aisi chiz likhdo jo sirf iss element ko represent kare, jaise firstpara. Jab ham ise yellow color ka background denge css ki madad se, tab ham iss element ka naam na de kar, kyuki iss page me 2 p tags hai aur hame particular iss p tag ke liye background yellow chahiye, ham likehnge '#firstpara' and will apply the desired styling(which we will cover in css later on).


Class

Class bhi ek attribute hi hai jo ki kissi bhi html element pe lagg sakta hai. ye use kiya jaata hai generally jab aapko kuch chuninda elements pe styling ya js manipulation karna hai. bohot saare elemnts ek saath ek class ke andar aa sakte hai but id kisi bhi elemnt ke liye unique hi hogi. An element can have multiple classes, jo ki jab aap uss element ke attribute me daleneg to space seperated honi chahiye.
Example ke liye agar ham iss html page pe css laga lete hai aur let's say ki ye jo paragraph chal raha hai ispe ham class lagana chahe to hame bas iss paragraph ke p tag me 'class' likhna hoga aur uski value aap kuch bhi likhdo, jaise 'scndpara'. Jab ham ise red color ka text denge css ki madad se, tab ham iss element ka naam na de kar, kyuki iss page me 2 p tags hai aur hame har paragraph ke liye red color text chahiye, ham likehnge '.scndpara' and will apply the desired styling(which we will cover in css later on).
