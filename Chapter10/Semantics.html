Header tag

This is header tag used to represent top section of the web page. Often contains logos, headings and navigation.
Nav tag signifies a navigation menu on a web page.
Aside tag is typically used for side bars
Main tag is used to represent main content of a website.
Article tag Indicates a self contained piece of content

Paragraph tag is used to write paragraphs about something.

Section tag represents a thematic grouping of contents on a webpage.
Footer tag usually contains the bottom-most part of the web page, like contact details and copyright information.
