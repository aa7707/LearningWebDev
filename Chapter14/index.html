Selectors is an alias of element tags used inside HTML. Using these selectors, one can add styling to the webpage. There are different types of selectors such as:-
Element Selectors
Used for styling using Elements of HTML. Syntax:- 
elementName{
    property: value;
    .
    .
}
Example:- 
body{
    background-color: black;
    color: white;
}
Class Selectors
Used for styling using Class name. It is used in situations where there are multiple elements, that can be different, that may require similar property or styling. These elements may have been grouped together in html file using class. Syntax:- 
.className{
    property: value;
    .
    .
}
Example:- 
.redBG{
    background-color: red;
    color: black;
}
ID Selectors
Used for styling using ID name. It is used in situations where there are multiple elements, that can be same, that may require different property or styling. These elements require to be seen distinctively in html file. Syntax:- 
#elementID{
    property: value;
    .
    .
}
Example:- 
#redBG{
    background-color: red;
    color: black;
}
Child Selectors
Used for styling an element, which is already inside other element. Element to be styled should be direct child of the parent element. Syntax:- 
ParentElement > ChildElement{
    property: value;
    .
    .
}
Example:- 
div > p{
    background-color: red;
    color: black;
}
Descendent Selectors
Used for styling an element, which is already inside other element. Element to be styled may not be direct child of the parent element. Syntax:- 
ParentElement DescendentElement{
    property: value;
    .
    .
}
Example:- 
div p{
    background-color: red;
    color: black;
}
Universal Selectors
Used for styling all elements of an html file. Syntax:- 
*{
    property: value;
    .
    .
}
Example:- 
*{
    margin: 10;
    padding: 40;
}
Pseudo Selectors
Used to style elements based on a particular state or user interaction. Syntax:- 
Element:Interaction{
    property: value;
    .
    .
}
Example:- 
a:hover{
    background-color: red;
}
a:link{
    color: black;
}
a:active{
    color: blue;
}
