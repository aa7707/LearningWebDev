CSS Sizing Units

There are various sizing units provided in CSS such as px,rem,em,vh,vw,% and more. These sizing units helps in better styling of page content. Let's understand them one by one.

PX-
PX is the abbreviated form of pixel. It is 196th of an inch. It is widely used unit.
REM-
REM applies the size decided for the content in the root element of the webpage.
EM-
EM applies the size decided for the content in the parent element of the webpage and multiplying it with value provided.
VH-
VH adjusts the size of the content to the Viewport height,i.e., height of the device in which the website is being opened.
VW-
VW adjusts the size of the content to the Viewport width,i.e., width of the device in which the website is being opened.
VMIN-
VMIN allows you to create responsive elements that adjust their size based on the smaller dimension of the viewport, whether it’s the width or height.
VMAX-
VMAX allows you to create responsive elements that adjust their size based on the bigger dimension of the viewport, whether it’s the width or height.
