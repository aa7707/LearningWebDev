CSS Shadows and Outlines

Now, we are going to learn about the shadow and the outline property in the CSS. These, as the name suggests, provide shadow and outlines for which they are applied.

Shadow

Shadows can be applied to anything in the web page from texts to boxes to scrolls etc. You just need to define the shadow property to any of them and it will be applied. Here is the syntax and example for both of that.
Syntax:- 
.classname {
    material-shadow: h-offset v-offset blur spread color;
}

material: here means either box, text or scrolls
h-offset: horizontal offset
v-offset: vertical offset
blur: larger the number, the blurier is the shadow
spread: area coverage around the material
color: any color of your choice

Example:-
<style>
    .box {
            border: 5px solid black;
            padding: 30px;
            box-shadow: -8px 11px 3px 5px red;
        }
    .ptext1 {
            text-shadow: 7px 7px 9px blueviolet;
        }    
</style>
<body>
{
    <div class="box">
        this is a box
    </div>

        quae assumenda commodi aliquam, mollitia harum quidem, sed suscipit adipisci earum est itaque! Tempora
        reiciendis magnam natus atque quidem vitae, reprehenderit ab itaque quis? Nobis animi neque adipisci!</p>
}
</body>

Outlines

Outlines are the definers of any space taken by the content including margin, borders, padding etc. It is applied over the border. It seems to be like a border, but isn't. It follows the border properties to specify it's content space. It is commonly used for highlighting focused elements.
Syntax:- 
.classname {
    outline: width style color;
}
width: width of the outline
style: dotted, solid or dashed style
color: sets the outline color

Example:- 

<style>
    .ptext2 {
            outline: 2px dotted red;
        }
</style>
<body>
    <p class="ptext2">Lorem ipsum, dolor sit amet consectetur adipisicing elit. Eveniet, cupiditate recusandae dolore
        quae assumenda commodi aliquam, mollitia harum quidem, sed suscipit adipisci earum est itaque! Tempora
        reiciendis magnam natus atque quidem vitae, reprehenderit ab itaque quis? Nobis animi neque adipisci!</p>
</body>
Output:-

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Eveniet, cupiditate recusandae dolore quae assumenda commodi aliquam, mollitia harum quidem, sed suscipit adipisci earum est itaque! Tempora reiciendis magnam natus atque quidem vitae, reprehenderit ab itaque quis? Nobis animi neque adipisci!
