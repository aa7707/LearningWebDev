CSS - Styling Lists

Whenever we create a navigation bar(which we will see later), most of the time we create the nav bar elements as list items under unordered list. We can beautify those too using some properties that we will discuss now:-

list-style-position
This is used to describe the position of your element markers, whether you want it to be inside the nav bar or not. We will undertsand this with an example:-
For Inside-
        <style>
            nav ul li{
                list-style-position: inside;
                background-color: blue;
            }
        </style>
        <body>
            <nav>
                <ul>
                    <li>Home</li>
                    <li>About</li>
                    <li>Contact</li>
                </ul>
            </nav>
        </body>

Output:-
  • Home
  • About
  • Contact

For Outside:- 
            <style>
            nav ul li{
                list-style-position: outside;
                background-color: blue;
                }
            </style>
            <body>
            <nav>
                <ul>
                    <li>Home</li>
                    <li>About</li>
                    <li>Contact</li>
                </ul>
            </nav>
            </body>

Output:-
  • Home
  • About
  • Contact

list-style-type
This is used to give the list items(nav elements) any specific type of markers like - roman capitals, roman smalls, disc, square, circle and many more. For example:- 
        <style>
            nav ul li{
                list-style-type: "😒";
                background-color: blue;
            }
        </style>
        <body>
            <nav>
                <ul>
                    <li>Home</li>
                    <li>About</li>
                    <li>Contact</li>
                </ul>
            </nav>
Output:-
  • Home
  • About
  • Contact

list-style-image
This is used to give the list items(nav elements) any kind of image that would be diplayed as markers using the image url. 
        <style>
            nav ul li{
                list-style-image: url("https://i.imgur.com/lfx13EX.jpeg");
                background-color: blue;
            }
        </style>
        <body>
            <nav>
                <ul>
                    <li>Home</li>
                    <li>About</li>
                    <li>Contact</li>
                </ul>
            </nav>
Output:-
  • Home
  • About
  • Contact

list-style
This serves as the combination of list-style-image, or list-style-type with list-style-position as one. Example is shown below:- 
            <style>
            nav ul li{
                list-style: circle inside;
                background-color: blue;
            }
        </style>
        <body>
            <nav>
                <ul>
                    <li>Home</li>
                    <li>About</li>
                    <li>Contact</li>
                </ul>
            </nav>
Output:-
  • Home
  • About
  • Contact
