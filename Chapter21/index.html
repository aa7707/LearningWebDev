Overflow in CSS

We have encountered many times in our lives like if there is a box of certain height and width, we tend to write any content in this limit only. But on website, let's suppose you want to write a blog post for instance, but you don't know how much you gonna write. Therefore defining a box of certain height will cause the text to overflow or come out of the defined box. Here comes Overflow property of the CSS to the recue. This property can be used to any predefined box where you want to write your blog content, to not overflow the words rather hide it within the box(if the content is more than the box area) and will provide you scroller option to see rest of the content.
