CSS - Position

In CSS, there is this property called position which helps in adjusting the position of any element in the website, about which we will see now:-
  1. Relative:-
    The element will be displaced from its position relative to its Original position.
  2. Absolute:-
    The element will be displaced from its position relative to the position of its positioned-ancestors.
  3. Fixed:-
    The element will be fixed to a position set by you and will not change even if scrolled.
  4. Sticky:-
    Same as fixed but position does not change until the original position is scrolled by the user. It is used for navigation tabs in various websites.
