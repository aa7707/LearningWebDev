CSS Variables

While we use CSS for styling our webpages, we might encounter situations when we repeatedly use same style on different elements or sometimes same elements multiple times. It is such a tidious process to do same thing over and over again. This is where Variables come-in. Variables just like any other programming language, are the developer defined keywords that are assigned to do a specific styling. These are also of 2 types such as global and local. Both Variable's declaration are shown below:-
Global: 
        :root{
            --variablename: stylingproperty;
        }
Local: 
        .classname{
            styling: value;
            --variablename: stylingproperty;
        }
Note: ":root" is used to assign global variable(s). "--"represents variable. Let's see with an example how this works: 
    <style>
        :root{
            --color: rgb(65, 106, 195);
            --seccolor: rgb(91, 44, 210);
        }
        *{
            margin: 0;
            padding: 0;
        }
        .nav{
            background-color: var(--color);
        }
        .container{
            background-color: var(--seccolor);
        }
        ul li{
            list-style: none;
            padding: 34px;
        }
        ul{
            display: flex;
        }
        ul li:first-child{
            --color: rgb(0, 74, 0);
            background-color: var(--color);
        }
    </style>
