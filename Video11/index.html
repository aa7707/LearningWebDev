In this page, we are going to learn about Entities and Code tags in html.
Entities tag
It enables you to display characters that are reserved in html or are not readily available on keyboard. These are used to represent Special Characters in a format that the browser can understand. They start with ampersand(&) and end with semicolon(;). Examples of html entities are:- 
            &lt; for <
            &gt; for >
            &amp; for &
            &nbsp; for non-breaking space
            &copy; for ©

Quotation tags
This tag is used to quote something from an article, and search engines understands the context of the text. It is common in textual content. There are 2 quotation tags, blockquote, for writing in a block and writing a lot, and q tag, for writing inline and less. Basic syntax:- 
Blockquote - 
                <blockquote cite = "source-url">
                    Quoatation tag here
                </blockquote>
Q tag - 
                <q cite = "source-url">Quoatation tag here</blockquote>
