Start of a new Journey with CSS

CSS stands for cascading style sheets. This is a language that helps decorate a website. We can say if website is a car, then HTML is its metal body and CSS is its overall looks and feel. Here onwards, we are going to explore CSS in more detail.
CSS can be added to an html file using style tag(Internal CSS), or using link tag(External CSS), or style element(Inline CSS). We will cover each of them one-by-one.
Style tag
Used under head tag. Selector,i.e. element, once defined, need not to be defined for its multiple occurences. Whatever times an element comes in an html file, the style is applied to it that many times. Basic Syntax:-
<style>
    element{
        property1: value;
        property2: value;
        .
        .
        .
    }
</style>
Example:- Like in this page, we have added black background color on the syntax above and example below, which is written under pre tag. That is executed as follows:- 
<style>
    pre{
       background-color: black;
       color: white; 
    }
</style>
