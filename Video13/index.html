Ways of styling using CSS

Inline CSS
Inline CSS is a way of adding styling to an element of an HTML page and is limited to that element only. Like the bold property given above in "Inline CSS". 

<dt style="font-weight: bold;">Inline CSS</dt>
Internal CSS
Used under head tag. Selector,i.e. element, once defined, need not to be defined for its multiple occurences. Whatever times an element comes in an html file, the style is applied to it that many times. This should be used only when the project is small. Basic Syntax:- 
<style>
    element{
        property1: value;
        property2: value;
        .
        .
        .
    }
</style>
Example:- Like in this page, we have added black background color on the syntax above and example below, which is written under pre tag. That is executed as follows:- 
<style>
    pre{
       background-color: black;
       color: white; 
    }
</style>
External CSS
In this, you create a css file in the same folder in which your html file is present. Add the styles you want, then link this css file to your html file using link element, inside head of HTML. It is considered a good practise to use this when there's a big project. Syntax to link CSS:- 

<link rel="stylesheet" href="style.css">
