Lists and its types

Lists, in HTML, are of three types - Ordered lists, Unordered Lists and Definition lists.
Ordered Lists:-
Lists that have numbering along side. These are often used as to show a no. of objects, steps etc. This type of lists can be ordered in these ways:-
  1. Numbering(1,2,3...)
  2. Roman Numbering(I,II,III,....)
  3. Roman Numbering(i,ii,iii,....)
  4. Alphabetical Capital(A,B,C...)
  5. Alphabetical Smalls(a,b,c....)
Unordered Lists:-
This kind of list does not have numbering system. It uses these to represent points:-
  • Discs - default
  • Squares
  • Circles
Definition Lists:-
This list is used to define some terms on a webpage. Its clear examples are the above description of lists and their types. For applying definition list:-
  1. Declare Definition list using 'dl' tag.
  2. Write the term you are about to describe or define in between 'dt' tag(opening and closing).
  3. Now write the definition of the term in between 'dd' tag.
Note:-
You can change the type of list to any other by using 'type' attribute. For Example 'type="___"'. You can write any of the below values at the place of '___'. (discs, circles and square can only be used for unordered list.)
  1. 1 for Numbered list.
  2. I for Roman numbering(Capital).
  3. i for Roman numbering(small).
  4. A for Alphabetical numbering(Capital).
  5. a for Alphabetical numbering(small).
  6. disc
  7. circle
  8. square
