Name Age Designatiom

As we can see here in the table above there are 3 columns - Name, Age and Designation. These column names are called as table heads and is specified using 'th' tag, right after declaring table. Next we have the data that are filled in the table like 'Ayush', '24' and Student. These are called table data and are declared using 'td' tag. As you can see 'Student' is taking 3 row spaces, this is done using rowspan attribute in 'td' tag
