Inline:-
These are the elements which takes only the space required to fit them. Examples are - span, anchor etc.
Block:-
These are the elements which takes whole width of the browser window even if it is not required. Examples are - p tag, div tag, etc.
To show the difference we are going to use css to add some background colors to these elements.
