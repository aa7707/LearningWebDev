IDs - They are attributes to any elements which are unique to any particular element. An ID can only refer to any one element tag, and no other element or same element tag can use it.

Classes - They are attributes to any elements which are not unique to any particular element. A class can be common to multiple same and different element tags.

An element can be styled using both id and classes in css. To go via ID you have to write like this - #id. To go via Class you have to write like this - .classname
